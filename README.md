# auto-qol
tera-proxy compilation module for automated functions

## Dependency
- `command` module

## Usage
- __skip-cutscene : `skip`__
  - Toggle on/off

## Config
- __`enableSkipMovie`__
  - Initialize skip movie function on/off
  - Default is `true`
